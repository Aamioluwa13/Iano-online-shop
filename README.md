# Next.js Tailwind Shop (Mobile-first)

Quick scaffold demonstrating a mobile-first e-commerce storefront with a sliding cart drawer and WhatsApp order button.

Setup

```bash
# from project root
npm install
npm run dev
```

Features

- Next.js + Tailwind CSS
- Mobile-first product grid
- Product detail modal with Add to Cart
- Sliding cart drawer showing items, quantities, total
- "Order via WhatsApp" button that formats the order and opens WhatsApp using `encodeURIComponent`

Brand colors: black and off-white (configured in tailwind.config.js)
